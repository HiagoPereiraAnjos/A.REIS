# A.Reis Empresarial

Landing page em React + Vite pronta para deploy na Vercel.

## Requisitos

- Node.js 20.x

## Rodar localmente

1. Instale as dependencias:
   `npm install`
2. Rode em desenvolvimento:
   `npm run dev`
3. Validar build de producao:
   `npm run build`

## Deploy na Vercel

1. Suba o projeto para um repositorio Git (GitHub/GitLab/Bitbucket).
2. Na Vercel, clique em **Add New Project** e importe o repositorio.
3. Confirme as configuracoes (detecta Vite automaticamente):
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Clique em **Deploy**.
